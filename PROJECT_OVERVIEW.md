# 宇宙迷航 - 项目概述

## 项目简介

"宇宙迷航"是一个交互式的三维宇宙可视化项目，使用Three.js构建，旨在以直观的方式展现从宏观的银河系到微观的地球地理特征的宇宙层次结构。该项目融合了科学准确性与艺术美感，让用户能够沉浸式地探索宇宙的奥秘。

## 项目架构

### 前端技术栈
- **Three.js**: 3D图形渲染引擎
- **WebGL**: GPU加速的图形渲染
- **Web Audio API**: 程序化音乐生成
- **原生JavaScript**: 无框架依赖
- **CSS3**: 动画与界面设计

### 数据结构
- 程序化生成的天体数据
- 真实的地理坐标系统
- 分层的场景管理系统

## 核心功能模块

### 1. 银河系可视化模块
- **黑洞系统**: 基于《星际穿越》科学模型的三维黑洞可视化
  - 事件视界
  - 光子球层
  - 扭曲吸积盘
  - 相对论性喷流
  - 引力透镜效应

- **恒星分布**: 遵循真实银河系结构的恒星分布算法

### 2. 太阳系模拟模块
- **行星系统**: 八大行星的真实轨道与自转模拟
- **物理规律**: 遵循开普勒定律的轨道运动
- **小天体系统**: 小行星带、彗星、柯伊伯带、奥尔特云

### 3. 地球地理模块
- **地形可视化**: 城市、山峰、湖泊、河流的精确标记
- **坐标系统**: 赤道、回归线、极圈、经纬网
- **实时定位**: GPS位置标记功能
