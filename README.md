# 宇宙迷航 - 三维宇宙可视化项目

一个基于Three.js的交互式宇宙可视化项目，展现从银河系到地球的多层级宇宙结构。

## 功能特性

### 1. 银河系层级
- 《星际穿越》风格的黑洞可视化
  - 事件视界（Event Horizon）
  - 光子球层（Photon Sphere）
  - 三维扭曲吸积盘
  - 相对论性喷流
  - 引力透镜效应光环
  - 体积雾效果
- 银河系恒星分布（螺旋臂结构）
- 太阳系标记点

### 2. 太阳系层级
- 真实比例太阳系统
  - 太阳（多层日冕结构）
  - 八大行星（含真实纹理）
  - 土星光环
  - 月球轨道
- 小行星带（主带、特洛伊群等）
- 彗星系统（轨道、离子尾、尘埃尾）
- 柯伊伯带
- 奥尔特云

### 3. 地球层级
- 高精度地球模型（真实地形纹理）
- 云层系统
- 主要城市标记（25个全球主要城市）
- 世界最高峰标记（含南北极）
- 主要湖泊标记
- 主要河流标记
- 地理坐标系统（赤道、回归线、极圈、经纬线）

## 交互功能

### 导航控制
- 鼠标悬停查看信息提示
- 点击进入下一层级
- 返回按钮回到上级层级
- 语言切换（中文/英文）

### 地球控制面板
- 旋转速度控制
- 云层透明度控制
- 暂停/继续旋转
- 重置视图
- 显示/隐藏功能：
  - 城市标记
  - 山峰标记
  - 湖泊标记
  - 河流标记
  - 赤道线
  - 经纬线
  - 回归线
  - 极圈线

### 位置标记功能
- 获取当前地理位置
- 在地球上标记位置
- 清除自定义标记

## 技术特点

### 三维渲染
- 基于Three.js的高性能WebGL渲染
- 程序化纹理生成
- 粒子系统模拟
- 球型纹理贴图

### 动画系统
- 分层动画控制
- 物理规律模拟（开普勒定律）
- 自适应性能调节

### 音频系统
- Web Audio API程序化音乐生成
- 三层音乐（银河系、太阳系、地球）
- 实时音量控制

## 使用方法

### 直接访问
打开 `index.html` 文件即可在浏览器中运行

### 本地服务器运行（推荐）
```bash
# 使用Python内置服务器
python -m http.server 8000

# 使用Node.js的http-server
npx http-server

# 使用Live Server插件（VSCode）
```

- 推荐使用现代浏览器

- 在较低性能设备上可减少粒子数量

## 开发说明

本项目使用纯前端技术栈（HTML/CSS/JavaScript），无需后端服务。
- Three.js v128
- OrbitControls用于相机控制
- Web Audio API用于音频生成
