# 宇宙模型 - Journey to the Universe

一个精美的交互式3D宇宙可视化应用，支持银河系、太阳系和地球三个层级的探索，具备中英双语切换功能。

## 特性

- 🌌 三层结构：银河系 → 太阳系 → 地球
- 🌍 逼真的地球模型，包含城市、山峰、湖泊、河流等地理要素
- 🌠 三维太阳系模型，包含真实的行星、小行星带、彗星、柯伊伯带等
- 🌑 《星际穿越》风格的黑洞模型
- 🌐 中英双语界面，一键切换
- 🎛️ 丰富的控制面板，可调节旋转速度、云层透明度等
- 📍 详细的地理标记，包含城市、山峰、湖泊、河流信息
- 🎮 交互式操作，支持鼠标控制视角和探索

## 技术栈

- Three.js - 3D图形渲染引擎
- HTML/CSS/JavaScript - 前端技术
- WebGL - 3D渲染技术
- 响应式设计 - 适配不同屏幕尺寸

## 使用方法

直接在现代浏览器中打开 `index.html` 文件即可使用。

### 控制说明

- **鼠标拖拽** - 旋转视角
- **鼠标滚轮** - 缩放
- **点击天体** - 进入下一层级或查看详细信息
- **语言切换** - 点击右下角按钮切换中英文
- **控制面板** - 调节各种视觉效果

### 层级导航

- **银河系** - 查看银河系和黑洞
- **太阳系** - 探索太阳系行星
- **地球** - 详细查看地球地理信息
