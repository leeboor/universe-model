# GitHub Pages 设置指南

按照以下步骤启用 GitHub Pages 以在线访问您的宇宙模型：

## 1. 访问仓库设置
- 前往 https://github.com/leeboor/universe-model
- 点击 "Settings" 选项卡

## 2. 配置 GitHub Pages
- 向下滚动到 "Pages" 部分
- 在 "Source" 下拉菜单中选择 "Deploy from a branch"
- 选择 "main" 分支和 "/ (root)" 文件夹
- 点击 "Save"

## 3. 等待部署
- GitHub Pages 需要几分钟时间部署
- 页面会显示部署状态

## 4. 访问您的网站
- 部署完成后，可通过以下地址访问：
- https://leeboor.github.io/universe-model

## 5. 自定义域名（可选）
- 如果您有自己的域名，可以在 "Custom domain" 部分设置

## 注意事项
- 首次部署可能需要 5-10 分钟
- 每次更新 main 分支时，GitHub Pages 会自动更新
- 确保主页面文件名为 index.html

## 故障排除
- 如果页面无法加载，请检查文件名是否为 index.html
- 确认 main 分支包含您的网站文件
- 检查是否有任何拼写错误
