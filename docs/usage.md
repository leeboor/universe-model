# 使用指南

## 快速开始

1. 下载 `index.html` 文件
2. 在浏览器中打开文件
3. 开始探索宇宙！

## 详细操作说明

### 银河系层级

在银河系层级，您可以看到：
- 《星际穿越》风格的黑洞模型
- 三维扭曲的吸积盘
- 喷流粒子系统
- 旋涡星系结构

**操作**：
- 点击黄色标记进入太阳系

### 太阳系层级

在太阳系层级，您可以看到：
- 太阳及其大气层结构
- 八大行星（包含真实纹理）
- 土星光环
- 小行星带
- 彗星系统
- 柯伊伯带
- 奥尔特云

**操作**：
- 点击地球进入地球层级
- 点击返回按钮回到银河系

### 地球层级

在地球层级，您可以看到：
- 高清地球纹理
- 云层系统
- 城市标记
- 山峰标记
- 湖泊标记
- 河流标记
- 经纬线、赤道、回归线等

**操作**：
- 悬停查看详细信息
- 点击标记查看详细资料
- 使用控制面板调节各种参数

## 控制面板功能

### 地球控制面板

- **旋转速度** - 调节地球自转速度
- **云层透明度** - 调节云层透明度
- **暂停旋转** - 暂停地球自转
- **重置视图** - 重置视角
- **显示/隐藏** - 控制城市、山峰、湖泊、河流等元素的显示

## 技术说明

应用使用了最新的WebGL技术，通过Three.js库实现3D渲染。所有数据和代码都集成在一个HTML文件中，便于分发和使用。
