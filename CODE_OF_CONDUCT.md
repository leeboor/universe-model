# 贡献者契约行为准则

## 我们的承诺

身为社区成员、贡献者和领袖，我们承诺使参与者不受骚扰，无论其年龄、身体状况、可见或不可见的缺陷、族裔、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、相貌、种族、宗教信仰或性取向如何。

我们承诺以有助于建立开放、友善、多样化、包容、健康社区的方式来行动和互动。

## 我们的准则

有助于为我们的社区创造积极环境的行为例子包括但不限于：

* 表现出对他人的同情和善意
* 对他人表达尊重，包括不同的观点、经验、观点和感受
* 提供和接受建设性反馈
* 承担责任并为错误道歉，从经验中学习
* 注重社区共同利益，而非个人利益

不当行为的例子包括：

* 使用性化语言或图像，以及不受欢迎的性关注或举动
* 恶意评论、人身或政治攻击
* 公开或私下的骚扰
* 未经明确许可，发布他人的私人信息，如物理或电子邮件地址
* 其他在专业场合不适当的行为

## 责任和权力

社区领袖有责任澄清和执行我们的行为准则，并将采取适当、公平的纠正措施回应任何他们认为不恰当、威胁、攻击性或有害的行为。

社区领袖有权和义务删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki编辑、议题和其他贡献，并在适当时机告知采取这些措施的原因。

## 适用范围

本行为准则适用于所有社区场所，也适用于在公共场所代表社区时的个人。

代表社区的情况包括使用官方电子邮件地址、通过官方社交媒体账号发布内容，或担任指定代表的角色。

## 执法

辱骂、骚扰或其他不可接受的行为可通过 [contact@universe-model.com](mailto:contact@universe-model.com) 向负责执法的社区领袖报告。

所有投诉都将得到及时和公平的审查和调查。

所有社区领袖都有义务尊重任何事件报告者的隐私和安全。

## 执法指导

社区领袖将遵循以下行为准则执法指导：

### 1. 纠正

**社区影响**: 使用不当语言或其他不专业或不受欢迎的行为。

**后果**: 由社区领袖发出非公开的书面警告，明确说明违规行为的性质，并解释为什么行为是不当的。可以要求公开道歉。

### 2. 警告

**社区影响**: 单次或重复违反社区准则的行为。

**后果**: 在指定时间内（通常为30天）警告不得与涉及的人员接触，也不得与事件有关的社区空间互动。在此期间，禁止通过社区渠道与他人进行任何形式的互动或交流——包括主动与行为准则执行者互动。这种禁止包括避免在社区空间和外部渠道进行互动。违反这些条款可能会导致临时或永久封禁。

### 3. 临时封禁

**社区影响**: 严重违反社区准则，包括持续的不当行为。

**后果**: 在指定时间内（通常为3个月到1年）禁止与社区进行任何形式的互动或公开交流。在此期间，不得与涉及的人员进行私人交流，也不得通过第三方进行交流。不得与事件有关的社区空间互动。违反这些条款可能会导致永久封禁。

### 4. 永久封禁

**社区影响**: 表现出暴力行为、恐吓行为、骚扰模式、持续不当行为，或直接影响社区安全的其他行为。

**后果**: 永久禁止在任何社区空间进行任何形式的公共互动。
